# Phase 13.5 Plan 01: HPLC Debug Log + Source File Persistence Summary

**One-liner:** Added `debug_log` JSON column to `hplc_analyses` + source file archive in `raw_data` for full HPLC analysis audit trail.

## What Was Built

Backend support for persisting two new audit trail artifacts on every HPLC analysis row:

1. **`debug_log` column** — stores `[{level, msg}]` entries capturing the full processing context from the frontend's `buildDebugLines()` call
2. **`raw_data` expansion** — when `source_files` is provided, the stored JSON includes both `source_files` (full file contents for reproduction) and `file_manifest` (filename + SHA256 + size for lightweight audit verification)
