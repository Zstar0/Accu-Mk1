### KPI card internal layout

Each KPI card uses `CardContent` with `py-4 px-4` padding (override default `py-6`):

```

    <p className="text-xs text-muted-foreground font-medium uppercase tracking-wide">{label}</p>
```

KPI icons: use lucide-react icons at `h-4 w-4 text-muted-foreground` placed before the label.

| KPI Card | Icon | Label | Value source |
|----------|------|-------|-------------|
| Open Worksheets | `ClipboardList` | "Open Worksheets" | count of worksheets with status `open` |
| Items Pending | `ListChecks` | "Items Pending" | sum of `item_count` across open worksheets |
| High Priority | `AlertTriangle` | "High Priority" | count of items with priority `high` or `expedited` across all worksheets |
| Avg Age | `Clock` | "Avg Age" | average ms since earliest `added_at` across open worksheets, formatted as `AgingTimer` string |

## Interaction Contract

### Filter interactions

| Interaction | Behavior |
|-------------|----------|
| Click status tab (All/Open/Completed) | Updates local `statusFilter` state → TanStack Query key changes → re-fetches from backend with `status` param |
| Select analyst from dropdown | Filters rendered rows client-side (post-fetch); does not trigger re-fetch |
| Select "All analysts" (default) | Clears client-side analyst filter; shows all rows for current status tab |

### Row click

Clicking any worksheet row: call `useUIStore.getState()` to get `setActiveWorksheetId` and `setWorksheetDrawerOpen`, set `activeWorksheetId = row.id`, set `worksheetDrawerOpen = true`. The list page remains visible behind the drawer overlay per D-12.

### Data polling

TanStack Query `refetchInterval: 30_000`, `staleTime: 0` — consistent with inbox pattern per CONTEXT.md D-13 and accumulated decisions.

### Loading state

While initial query is loading (`isLoading === true`):
- KPI row: 4 `<Skeleton className="h-[72px] rounded-xl" />` cards in 4-column grid
- Table: 5 `<Skeleton className="h-10 w-full" />` rows inside the table body

While re-fetching in background (not initial load): show data as-is; no loading overlay.
