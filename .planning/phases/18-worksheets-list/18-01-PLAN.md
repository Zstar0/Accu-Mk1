**KPI computation (per D-01, D-02 — all client-side from full response):**
Compute from the UNFILTERED `worksheets` array (so KPIs reflect global state regardless of analyst filter). Use `useMemo`:
- `openCount`: `worksheets.filter(w => w.status === 'open').length`
- `itemsPending`: `worksheets.filter(w => w.status === 'open').reduce((sum, w) => sum + w.item_count, 0)`
- `highPriorityCount`: `worksheets.flatMap(w => w.items).filter(i => i.priority === 'high' || i.priority === 'expedited').length`
- `avgAge`: Compute average ms since each open worksheet's earliest `added_at`:
  ```typescript
  const openWithItems = worksheets.filter(w => w.status === 'open' && w.items.length > 0)
  const now = Date.now()
  const ages = openWithItems.map(w => {
    const earliest = w.items
      .map(i => i.added_at ? new Date(i.added_at).getTime() : now)
      .reduce((min, t) => Math.min(min, t), now)
    return now - earliest
  })
  const avgMs = ages.length > 0 ? ages.reduce((s, a) => s + a, 0) / ages.length : 0
  ```
  Format avgMs as human-readable: hours and minutes, e.g. `"4h 23m"`. If 0, display `"—"`.
