<objective>
Use standard injection RTs for identity comparison when available, falling back to calibration curve reference_rt.

Purpose: Standard injections run on the same HPLC method give accurate RT comparison (delta ~0.02 min). Calibration curve RTs may be from a different method, causing false "DOES NOT CONFORM" results (delta ~6+ min). This plan makes identity checks method-aware.

Output: Modified analysis pipeline that prefers standard injection RTs and records the reference source in the calculation trace.
</objective>
