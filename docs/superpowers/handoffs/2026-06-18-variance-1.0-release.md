## Critical operational gotchas

| Gotcha | Why it matters | How to handle |
|---|---|---|
| **IS has no `--reload`** | The additional COA rendered bare because the live IS predated the H5 commit ‚Äî restarting backend+COABuilder but NOT the IS was the bug | `docker restart integration-service` after ANY IS code change. (Same for `accu-mk1-backend`, `coabuilder-varcap-live`.) |
| **`:5000` runs `coabuilder-varcap-live`, NOT the baked image** | Branch COABuilder 2.27.0 was made live by stopping `coabuilder_service` and running a sibling container from the same image with `C:/tmp/coabuilder-varcap` bind-mounted at `/app` | **REVERT after UAT:** `docker rm -f coabuilder-varcap-live && docker start coabuilder_service`. Recipe in memory `architecture_coabuilder_container_topology`. |
| **docker.exe needs Windows-style paths** even under `MSYS_NO_PATHCONV=1` | `--env-file /c/tmp/x` fails; `-v /c/tmp/x` fails | Use `C:/tmp/x` for docker `-v`/`--env-file`; `/c/tmp/x` only for bash builtins |
| **NEVER hardcode SENAITE_PASSWORD** into a heredoc/env-file | Auto-mode classifier denies it (credential leak) | Machine-copy env from a running container: `docker exec <c> printenv \| grep -vE '...' > ENV`, use `--env-file`, then `rm ENV` |
| **Conflict resolver: line-based, not regex** | A DOTALL regex spans across adjacent/empty-side conflict blocks and corrupted the php `if/endif` balance | Use a line-based state machine (keep ours / drop theirs / drop markers) + `php -l` after |
| **Mk1 backend tests run against the SHARED live DB** | ~19 suite failures are pre-existing/env-flaky (clickup/sla/priorities/completion/peptide-e2e) + polluted by this session's manual BW-0016/ZZTEST data ‚Äî NOT regressions | All dedicated variance/coa suites are green; treat the 19 as stale/env (user's additive-only principle) |
| **"Verified ‚Äî Variance" vs "blue chip"** | Row-level chip (AnalysisTable ~1364, still `showVarianceChip`) intentionally hides on signed-off rows (they show the badge); only the SOURCE-vial chips were changed to always-blue | Don't "fix" the row-level chip to match ‚Äî by design |
| BW COA needs a Sample image + Chromatogram attachment | Generation stops at "Attaching to SENAITE" without them ‚Äî by-design gate, unrelated to variance | Attach via the sample page pickers before generating |
